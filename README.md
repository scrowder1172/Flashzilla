# Project 17: Flashzilla

## Hacking With Swift project 17

This is Project 17 in Paul Hudson's 100 Days of SwiftUI course. The project creates a flashcard iOS app to help the user study for tests, learn a new langauge, etc.

## Course Link

Check out Paul's course for more on SwiftUI: https://www.hackingwithswift.com/100/swiftui

## How to install

1. fork the project or download the .zip
2. open project in XCode v15 or later (earlier versions may or may not compile correctly)
3. run in simulator
